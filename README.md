# TTWeather

Telegram-Bot

# How to use ?

```bash
go mod init TTWeather
go mod tidy
go get -u gopkg.in/tucnak/telebot.v2
```

## Can only monitor private chat messages, but not group messages?

1. Find @BotFather and send `setprivacy`;
2. Enter the name of the bot, such as `@XXBot`;
3. Reply to Disable.
